# TradeProspect Pro - 外贸客户开发系统

> 专为中小外贸企业打造的一站式客户开发与管理平台

## 🚀 快速启动

```bash
cd trade-prospect-pro
npm install
npm run dev
```

访问 http://localhost:5173/ 即可使用

## 📋 功能概览

| 模块 | 功能 |
|------|------|
| 🏠 仪表盘 | KPI统计、趋势图表、销售漏斗、待办事项 |
| 🔍 找客户 | 海关数据、企业图谱、LinkedIn挖掘、地图拓客 |
| 👥 客户管理 | CRM客户库、Pipeline管理、跟进记录 |
| ✉️ 邮件中心 | 邮件模板、群发、追踪、自动化序列 |
| 📅 任务日程 | 任务管理、日历视图、提醒设置 |
| 📊 数据分析 | 增长趋势、分布图表、效果漏斗 |
| ⚙️ 系统设置 | 账号、团队、邮箱配置、数据导入导出 |

## 🛠 技术栈

- React 18 + TypeScript
- Ant Design 5.x
- Recharts
- React Router 6
- Vite
